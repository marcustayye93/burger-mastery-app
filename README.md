# Burger Mastery App v0.1.0

Sprint 1 production foundation.

## Included
- Mobile-first PWA shell
- Home, Learn, Builder preview, Lab preview and Profile
- Persistent five-tab navigation
- Light/dark appearance
- Swipe-style educational stories
- Direct integration with approved assets from `burger-mastery-assets`
- Local progress state
- Service worker and installable manifest

## Deploy
Upload all files to the root of a public GitHub repository. Enable GitHub Pages from `main` and `/ (root)`.
